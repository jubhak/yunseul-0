<h2>텍스트 추출 프로그램 - 윤슬 제로 V1.8</h2>
<img width="664" height="718" alt="image" src="https://github.com/user-attachments/assets/09fad365-daa9-48f5-b707-fc511f10477f" />
<ul>
  <li>텍스트로 되어 있는 값들에 대해 추출 키 값을 입력하여 데이터 추출, 엑셀로 자동 등록</li>
  <li>콤마 기준, 중복으로 추출 키 값을 입력할 수 있으며 선 조건으로 추출되지 않으면 콤마 기준 그 다음 키워드로 추출</li>
  <li>1., 과 같은 규칙은 자동으로 가장 후순위 조건으로 적용. 키 값 상관 없이 1. 식으로 시작하는 값을 추출</li>
  <li>Col No. 에는 엑셀에 입력할 컬럼 순서 정의</li>
  <li>자동 입력할 엑셀파일 선택 시, 가지고 있는 엑셀시트 자동 리스트업, 입력 시트 선택 가능</li>
  <li>Unique 설정 시, 해당 컬럼 위치에 동일한 값이 있을 경우 마지막 줄에 입력이 아닌 기존 데이터 업데이트</li>
</ul><br />
