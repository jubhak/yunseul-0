## 1. 프로그램 개요

YUNSEUL-0은 아래와 같은 비정형 연락처/인적 정보 텍스트를 붙여넣으면, 키워드 기반으로 각 항목을 자동 추출하여 지정된 엑셀 파일의 원하는 컬럼에 입력해주는 도구입니다.

**입력 예시:**
```
(L2-001)
Networker Name: John
1. Full Name: Kim Minsoo
2. Gender: Male
3. Birthday: 1990-05-15
4. Contact: 010-1234-5678
5. Occupation: Engineer
6. City of Residence: Seoul
7. Church Name: Grace Church
8. Denomination: Presbyterian
9. Church Position: Deacon
10. Married/Single: Married
11. Facebook address: facebook.com/minsoo
```

**처리 결과:** 각 항목이 자동으로 분리되어 테이블에 표시되고, 엑셀 파일에 한 번에 입력됩니다.

---

## 2. 시스템 요구사항

| 항목 | 요구사항 |
|------|----------|
| 운영체제 | Windows 10 이상 (64비트) |
| 추가 설치 | 없음 (자체 포함 실행 파일) |
| 엑셀 파일 | `.xlsx` 또는 `.xls` 형식 |

> 별도의 .NET 런타임 설치가 필요 없습니다. 프로그램 파일 하나로 바로 실행 가능합니다.

---

## 3. 설치 및 실행

1. `DataParser.exe` 파일을 원하는 폴더에 복사합니다.
2. `DataParser.exe`를 더블클릭하여 실행합니다.
3. 별도의 설치 과정은 없습니다.

> 처음 실행 시 Windows SmartScreen 경고가 나타날 수 있습니다. **"추가 정보"** → **"실행"**을 클릭하면 됩니다.

## 4. 화면 구성

프로그램은 크게 **3개 영역**으로 나뉩니다.

### 4.1 상단 — DATA INPUT (데이터 입력 영역)

```
┌──────────────────────────────────────┐
│  [▓]  YUNSEUL-0                   ✕  │  ← 타이틀 바 (드래그로 창 이동)
├──────────────────────────────────────┤
│  ▌ DATA INPUT                        │
│ ┌──────────────────────────────────┐ │
│ │                                  │ │  ← 텍스트 입력 영역
│ │  (여기에 원본 데이터를 붙여넣기)    │ │
│ │                                  │ │
│ └──────────────────────────────────┘ │
│          [ ▶  1차 가공 ]             │  ← 파싱 실행 버튼
└──────────────────────────────────────┘
```

- **텍스트 입력 영역**: 파싱할 원본 데이터를 붙여넣는 곳입니다.
- **▶ 1차 가공 버튼**: 입력된 텍스트를 키워드 기반으로 파싱합니다.

### 4.2 중간 — PARSED RESULT (파싱 결과 테이블)

```
┌──────────────────────────────────────┐
│  ▌ PARSED RESULT  [+]               │  ← [+] 행 추가 버튼
├──────────────┬───────────┬───────────┤
│  KEYWORDS    │  VALUE    │  COL #    │  ← 헤더
├──────────────┼───────────┼───────────┤
│ ID           │ L2-001    │  1        │
│ Networker... │ John      │  2        │
│ Full Name... │ Kim Min.. │  3        │
│ Gender       │ Male      │  4        │
│  ...         │  ...      │  ...      │
└──────────────┴───────────┴───────────┘
```

| 컬럼 | 설명 |
|------|------|
| **KEYWORDS** | 파싱에 사용할 키워드 목록 (쉼표로 구분, 편집 가능) |
| **VALUE** | 파싱 결과로 추출된 값 (편집 가능) |
| **COL #** | 엑셀에 입력할 컬럼 번호 (1 = A열, 2 = B열, ...) |

### 4.3 하단 — 엑셀 연동 영역

```
┌──────────────────────────────────────┐
│ [📂 엑셀파일 선택]  선택된 파일 없음   │
│ 엑셀시트 선택  [▼ 시트 드롭다운    ]  │
│ ─────────────────────────────────── │
│          [ ⬇  입력처리 ]             │  ← 엑셀에 데이터 삽입
└──────────────────────────────────────┘
```

- **📂 엑셀파일 선택**: 데이터를 입력할 엑셀 파일을 선택합니다.
- **시트 드롭다운**: 엑셀 파일 내 시트를 선택합니다.
- **⬇ 입력처리**: 파싱된 데이터를 엑셀에 삽입합니다.

## 5. 사용 방법 (단계별)

### Step 1: 엑셀 파일 준비 (최초 1회)

1. 하단의 **[📂 엑셀파일 선택]** 버튼을 클릭합니다.
2. 데이터를 입력할 `.xlsx` 파일을 선택합니다.
3. 파일명이 버튼 옆에 표시되고, 시트 목록이 드롭다운에 로드됩니다.
4. 원하는 **시트**를 드롭다운에서 선택합니다.

> 한 번 선택하면 다음 실행 시 자동으로 기억됩니다.

### Step 2: COL # (컬럼 번호) 설정 (최초 1회)

결과 테이블의 **COL #** 열에 각 항목이 들어갈 엑셀 컬럼 번호를 입력합니다.

| COL # 값 | 엑셀 컬럼 |
|-----------|-----------|
| 1 | A열 |
| 2 | B열 |
| 3 | C열 |
| ... | ... |
| 26 | Z열 |

> 컬럼 번호는 숫자만 입력 가능하며, 최대 3자리까지 지원합니다.
> 한 번 설정하면 다음 실행 시 자동으로 기억됩니다.

### Step 3: 데이터 입력

1. 상단 **DATA INPUT** 영역에 원본 텍스트 데이터를 **붙여넣기(Ctrl+V)** 합니다.
2. **[▶ 1차 가공]** 버튼을 클릭합니다.
3. 중간 테이블의 **VALUE** 열에 파싱 결과가 자동으로 채워집니다.

### Step 4: 결과 확인 및 수정

- VALUE 열의 값을 직접 클릭하여 **수정**할 수 있습니다.
- 잘못 파싱된 항목이 있으면 수동으로 고쳐주세요.

### Step 5: 엑셀에 입력

1. 엑셀 파일이 **닫혀 있는지** 확인합니다. (열려 있으면 저장 불가)
2. **[⬇ 입력처리]** 버튼을 클릭합니다.
3. 데이터가 선택한 시트의 **마지막 데이터 행 다음 행**에 자동 삽입됩니다.
4. 완료 메시지가 표시되면 성공입니다.
5. 입력 완료 후 텍스트 입력 영역과 VALUE 값이 자동으로 초기화됩니다.

### Step 6: 반복

다음 데이터를 붙여넣고 Step 3~5를 반복합니다.

## 6. 키워드 설정 가이드

### 6.1 키워드 작동 원리

KEYWORDS 열의 값은 입력 텍스트에서 해당 항목을 찾기 위한 **검색어** 역할을 합니다.

- 쉼표(`,`)로 구분하여 **여러 키워드**를 지정할 수 있습니다.
- 키워드 중 하나라도 매칭되면 해당 값을 추출합니다.

### 6.2 키워드 유형

#### 텍스트 키워드
콜론(`:`) 앞의 항목명과 매칭합니다.

```
키워드: "Full Name, name"
→ "Full Name: Kim Minsoo" 에서 "Kim Minsoo" 추출
→ "Name: Park Jihye" 에서도 "Park Jihye" 추출
```

#### 번호 패턴 키워드
`숫자.` 형식으로 번호 기반 매칭을 합니다.

```
키워드: "1., Full Name, name"
→ "1. Full Name: Kim Minsoo" 에서 "Kim Minsoo" 추출
→ 번호 매칭과 텍스트 매칭을 동시에 시도합니다
```

### 6.3 기본 키워드 목록

| 행 | 기본 키워드 |
|----|------------|
| 1 | `ID` |
| 2 | `Networker Name, networker` |
| 3 | `1., Full Name, name` |
| 4 | `2., Gender` |
| 5 | `3., Birthday, date of birth, dob` |
| 6 | `4., Contact, phone, mobile` |
| 7 | `5., Occupation, job` |
| 8 | `6., City of Residence, city, residence` |
| 9 | `7., Church Name, church` |
| 10 | `8., Denomination` |
| 11 | `9., Church Position, position` |
| 12 | `10., Married/Single, marital status` |
| 13 | `11., Facebook address, facebook, fb` |

### 6.4 행 추가/삭제

- **행 추가**: PARSED RESULT 옆의 **[+]** 버튼을 클릭합니다.
- **행 삭제**: 키워드가 비어있는 행은 **[⬇ 입력처리]** 시 삭제 여부를 묻는 대화상자가 나타납니다.

## 7. 설정 저장 및 복원

프로그램은 아래 항목을 **자동으로 저장**합니다:

| 저장 항목 | 저장 시점 |
|-----------|-----------|
| 엑셀 파일 경로 | 파일 선택 시 |
| 키워드 (KEYWORDS) | 키워드 편집 후 포커스 이동 시 |
| 컬럼 번호 (COL #) | 번호 편집 후 포커스 이동 시 |
| 행 개수 | 행 추가/삭제 시 |

- 설정 파일 위치: 프로그램과 같은 폴더의 `settings.json`
- 프로그램 재실행 시 마지막 설정이 자동으로 복원됩니다.

## 8. 자주 묻는 질문 (FAQ)

### Q: 창을 어떻게 이동하나요?
**A:** 타이틀 바(`[▓] YUNSEUL-0`), 라벨 영역, 또는 빈 공간을 마우스로 드래그하면 창이 이동합니다.

### Q: 창을 닫으려면?
**A:** 우측 상단의 **✕** 버튼을 클릭합니다.

### Q: 프로그램이 항상 최상위에 표시되는데 끌 수 있나요?
**A:** 현재 버전에서는 항상 최상위(Always on Top) 모드가 기본입니다. 다른 프로그램의 데이터를 복사-붙여넣기 하기 편하도록 설계되었습니다.

### Q: 엑셀 파일이 열려 있어도 되나요?
**A:** 아닙니다. **[⬇ 입력처리]** 시 엑셀 파일이 다른 프로그램(Excel 등)에서 열려 있으면 저장에 실패합니다. 반드시 파일을 닫은 후 시도해주세요.

### Q: 데이터가 엑셀 어디에 입력되나요?
**A:** 선택한 시트에서 데이터가 있는 **마지막 행의 바로 다음 행**에 입력됩니다. 기존 데이터를 덮어쓰지 않습니다.

### Q: COL # 을 비워두면 어떻게 되나요?
**A:** COL # 이 비어있거나 0인 행은 엑셀에 입력되지 않고 건너뜁니다.

### Q: Facebook 관련 텍스트가 자동으로 제거되나요?
**A:** 네. 입력 데이터에서 `Facebook Add` 관련 문구는 파싱 전에 자동으로 제거됩니다.

## 9. 문제 해결

### 프로그램이 실행되지 않는 경우
- Windows 64비트 환경인지 확인합니다.
- 바이러스 백신이 차단하고 있지 않은지 확인합니다.
- 프로그램 폴더에 쓰기 권한이 있는지 확인합니다.

### "엑셀 파일이 열려 있어 수정할 수 없습니다" 오류
- Microsoft Excel 또는 다른 프로그램에서 해당 파일을 닫습니다.
- 작업 관리자에서 Excel 프로세스가 남아있지 않은지 확인합니다.

### 파싱 결과가 비어있는 경우
- 입력 데이터의 형식이 `항목명: 값` 또는 `번호. 항목명: 값` 형태인지 확인합니다.
- KEYWORDS 열의 키워드가 입력 데이터의 항목명과 일치하는지 확인합니다.
- 키워드를 쉼표로 구분하여 여러 변형을 추가해보세요.

### 프로그램 오류 발생 시
- 프로그램 폴더에 `crash.txt` 파일이 생성됩니다.
- 해당 파일의 내용을 개발자에게 전달해주세요.
